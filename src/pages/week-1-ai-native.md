This is my first public learning log.

The main lesson is simple: building in public starts with shipping small artifacts.
# Why I Started
Học hỏi thêm về AI basic, các platform AI + agent thường dùng phục vụ cho ecosystem như Hermes trong tương lai. Học kiến thức nền tảng + cân đối với thực hành để đảm bảo đủ skills vận hành product lâu dài

# What I Learned
This week, I set up Python, Git, GitHub, an Astro blog, and deployed it with Vercel. Most important thing is read some technical blogs about AI: Hamel Husain blog "LLM Evals FAQ" + "Your AI Product Needs Evals" and Eugene Yan blog · "Patterns for Building LLM Systems"

# Friction
